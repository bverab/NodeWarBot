# Spectre Dashboard Assets

Use these folders for production-ready dashboard visuals only.

- `brand/` - Spectre logos, glyphs, and wordmarks.
- `characters/` - original Spectre mascot or character artwork.
- `backgrounds/` - hero, dashboard, and atmospheric background images.
- `illustrations/` - feature and workflow illustrations.
- `icons/` - custom icons that are not provided by the UI icon library.

Do not place copyrighted reference images here unless they are explicitly licensed for production use.
